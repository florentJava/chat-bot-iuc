# chat-bot-iuc
chatbot de renseignement de l'institut sur le formation de l'institut universitaire de la cote universitaire de la cote


## installer les outils 

    - installer python

    - installer telebot :
            pip install telebot
    - installer pandas : 
            pip install pandas

    - demarer le bot :
            python index2.py

##  Utiliser le bot
    - aller sur telegramme et rechercher : iuc-infos
